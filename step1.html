Step 1

Download and setup the data

The first step for my project was to download and set up the data. The data I got was from the VDOT website, and it was in a .csv file format. The data had different variables such as, what location the crash happened (the latitude and longitude coordinates), the crash date (When the crash happened including the time), what part of the vehicle was damaged, the weather, etc.

 

I then set up the data by writing some code that would filter out the data. So it only chose the important variables like: "CRASH_SEVERITY", "COLLISION_TYPE", WEATHER_CONDITION", "ROADWAY_DESCRIPTION", "INTERSECTION_TYPE", "WORK_ZONE_LOCATION", "WORK_ZONE_TYPE", "SCHOOL_ZONE". There were a lot of other variables but these are just a few of them.
